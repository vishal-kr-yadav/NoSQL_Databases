**Installation:**

+ **Solr 7.4.0**  
    - wget http://archive.apache.org/dist/lucene/solr/7.4.0/solr-7.4.0.tgz
    - tar xzf solr-7.4.0.tgz
    - cd solr-7.4.0
    - bin/solr start
    - Create solr core
        - cd sol-7.4.0/bin
        - ./solr create_core -c <**core_name**>
        
+ **pysolr**  
    - pip install pysolr
